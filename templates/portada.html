VIDA SALUDABLE

Bienvenido a nuestro proyecto “Vida Saludable”, una página web creada para mostrar la importancia de la alimentación, el descanso y la actividad física en nuestra vida diaria. Aquí encontrarás información relacionada con Historia, Química y una calculadora de IMC, combinando tecnología y aprendizaje de una forma práctica e interactiva.

Nuestro Proyecto

Conoce los aspectos principales del proyecto.

Objetivo

Crear una interfaz gráfica en Python conectada a una base de datos para mostrar información sobre alimentación saludable, descanso, actividad física y química del cuerpo.

Funcionamiento

Nuestro proyecto es una interfaz gráfica en Python con Flask que incluye apartados de Historia, Química y una calculadora de IMC conectada a una base de datos para almacenar y mostrar resultados de forma visual.

Impacto Social

Este proyecto busca crear conciencia sobre la importancia de la alimentación, el descanso y la actividad física, motivando a los estudiantes a mejorar sus hábitos y cuidar su salud.

Reflexión

Este proyecto nos permitió relacionar la tecnología con la salud, aplicando conocimientos de distintas materias para crear una página web útil y comprender la importancia de mantener hábitos saludables.
